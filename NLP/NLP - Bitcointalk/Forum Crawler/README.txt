How to run
1. Install the necessary module using pip -r install requirements.txt.
2. Write a config file for either purpose.
  2-1. To crawl a single page, refer to the crawling-page-config-template.json file and write one (excluding the notes).
  2-2. To crawl multiple pages, refer to the crawling-pages-config.template.json file and write one (excluding the notes).
3. Start crawling with python app.py CONFIG_FILE_NAME.json.
